Musical Instrument Assignment

The goal of this assignment was to create a musical instrument using the Audrino Uno. After a lot of hardwork, I've created an instrument simular to a xylophone. Each induvidual key, which are made with paper clips are lined up across the bread board. When the positively charged paperclip hits one of the keys, a noise is made.
